# 🎯 The Big Goal
Build a **real-world** simulation where an AI agent can learn to solve a task.

### 🗳️ What to Expect
When Round 1 starts, you'll choose **1 of 4 or 5 problem statements**. You will build your OpenEnv environment based on that choice.

### 🚫 NO Games or Toys (Usually)
While some examples might say "mini-game", the general rule is to build something people actually do at work:
* 📧 Sorting emails
* 🖥️ Reviewing code
* 🧹 Cleaning up data
* 🗓️ Setting up meetings
* 🎧 Helping customers

### 🛠️ The Spec
You must use the **OpenEnv** standard. This means your code needs 3 main "buttons" for the AI to press:
1. `reset()` - Start over with a fresh environment.
2. `step()` - Make a move (the AI takes an action).
3. `state()` - Check how things are looking right now.
