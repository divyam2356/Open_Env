# 🏆 How to Win (Scoring)

| What matters | Weight | What are they looking for? |
| :--- | :--- | :--- |
| **Real utility** | **30%** | **Task Design**: Is it clear, realistic, and testable? |
| **Task Quality** | **25%** | **Grading Logic**: Does the reward system make sense? |
| **Design** | **20%** | **Interface**: Does it follow the OpenEnv standard? |
| **Code Quality**| **15%** | **Runtime**: Does it run without errors? |
| **Creativity** | **10%** | **Unique Idea**: Is it a cool new approach? |

### ⚠️ Instant Failures (Don't do these!)
* Environment doesn't start or answer.
* Copying someone else's work.
* Giving the same score every time.
* Forgetting the `inference.py` script.
